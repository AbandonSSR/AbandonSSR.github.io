Windows_10_11下载免费的 HEVC_Video_Extension

发表于 2023-07-04 更新于 2023-07-05 分类于 系统维护 本文字数： 250 阅读时长 ≈ 1 分钟

HEVC（High Efficiency Video Coding）是一种视频压缩标准，能够以较小的文件大小进行高质量的视频播放。在某些情况下，用户可以从 Microsoft Store 下载 HEVC 视频扩展。但是，从 Microsoft Store 下载文件可能有点棘手。幸运的是，Adguard 是一个第三方 Web 服务和在线链接生成器，可以轻松地从 Microsoft Store 下载 appx、appxbundle 和 msixbundle 文件。在本文中，我们将指导你使用 Adguard 下载 HEVC 视频扩展的。

解析链接

  1. 打开 Adguard 网站。链接：https://store.rg-adguard.net。
  2. 将HEVC视频扩展应用URL复制到文本框中。链接：https://www.microsoft.com/en-us/p/hevc-video-extensions-from-device-manufacturer/9n4wgh0z6vhq
  3. 点击右侧的对号

下载安装包

点击后缀为 .appxbundle 后缀的文件链接下载文件

安装

双击下载的安装包，点击安装，稍等片刻
